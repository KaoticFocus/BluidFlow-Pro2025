# Favicon Generation

The favicon files (`favicon.png` and `apple-touch-icon.png`) should be generated from `favicon.svg`.

## Quick Generation

1. Open `generate-favicons.html` in a browser
2. The PNG files will be automatically downloaded
3. Move them to the `public/` directory

## Manual Generation

You can also use any image editor or online tool to convert `favicon.svg` to:
- `favicon.png` (64x64)
- `apple-touch-icon.png` (180x180)
