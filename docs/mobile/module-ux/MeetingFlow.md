# MeetingFlow — Mobile UX (Next.js)

Routes
- /meetings (list)
- /meetings/[id] (record/summary)

List
- Upcoming/past tabs (segmented control). Cards show date/time, attendees, status.

Record/summary
- Primary: Start/Stop, Upload Transcript.
- Show summary, actions: share, pin, create tasks.

Acceptance
- Controls ≥ 44×44; sticky footer for primary action on small screens.
