# TaskFlow — Mobile UX (Next.js)

Routes
- /tasks (list)
- /tasks/[id] (detail)

List (mobile)
- Cardized items with title, due/date chip, status pill, chevron.
- Actions in overflow menu (…): complete, edit, delete.

Detail (mobile)
- Sticky header: back, title, primary action.
- Sections: description, checklist (44px items), activity.

Empty state
- Illustration + "Create your first task" CTA.

Acceptance
- No horizontal scroll at 360px; all actions reachable with one hand.
